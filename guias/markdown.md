# Guía de Markdown

Markdown es la sintaxis que usa Discord para darle formato al texto.  
Fun fact: ¡Esta página web está hecha en su mayoría con Markdown!

## Estilos de texto

- *Cursiva*
    - `*Texto*`
    - `_Texto_`
- **Negrita**
    - `**Texto**`
- ***Negrita cursiva***
    - `***Texto***`
- <u>Subrayado</u>
    - `__Texto__`
